# 📅 2026 Excel Planner Template

A clean and structured 2026 weekly planner built entirely in Microsoft Excel.

This template is designed to be minimal, distraction-free, and easy to use.  
The layout is protected to prevent accidental edits while keeping designated sections fully editable.

---

## ✨ Features

- Full 2026 calendar structure  
- Weekly planning layout  
- Dedicated task and notes sections  
- Clearly marked editable cells  
- Print-friendly format  
- Lightweight and fast  

---

## 🔒 Protection & Editing

- Only selected cells are editable  
- Formulas and structure are locked  
- Designed to prevent accidental formatting issues  

> Note: Excel sheet protection prevents casual edits but is not encryption-level security.

---

## 📥 How to Use

1. Download the file from this repository  
2. Open in Microsoft Excel (2019 or 365 recommended)  
3. Enter tasks only in the editable sections  
4. Save your own working copy  

If saved as `.xltx`, Excel will automatically create a new file when opened.
